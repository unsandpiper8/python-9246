## installation and usage:
if you for some reason wanted to install and run one of my scripts you need to install [uv](https://docs.astral.sh/uv/):   
with curl:
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```
with wget:
```
wget -qO- https://astral.sh/uv/install.sh | sh
```
with homebrew:
```
brew install uv
```
with winget:
```
winget install --id=astral-sh.uv  -e
```
(there are more ways to install uv explained at [uvs website](https://docs.astral.sh/uv/getting-started/installation/#pypi))

then clone the repo:
```
git clone https://github.com/unsandpiper8/python-9246.git
```
(or just manually download the desired script thru github)

### run script:
```
uv run (path/to/script.py)
```
